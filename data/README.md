# Data

Place the final labeled dataset here as CSV or JSON.

Suggested columns:

| Column | Purpose |
| --- | --- |
| `id` | Stable row id |
| `text` | Post or comment text |
| `label` | Final human label |
| `split` | `train`, `validation`, or `test` |
| `source` | URL or community reference, if appropriate |
| `notes` | Optional annotation notes |
